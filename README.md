# Cafe Vibes

Projeto de um site com tema de café desenvolvido para praticar HTML e CSS e criar uma interface visual agradável inspirada em cafeterias.

## Sobre o projeto

O **Cafe Vibes** é uma landing page simples que apresenta um layout moderno com foco em estilo visual, cores quentes e organização dos elementos na página.

O objetivo do projeto foi praticar conceitos de desenvolvimento front-end e criação de interfaces.

## Tecnologias utilizadas

* HTML
* CSS

## Funcionalidades

* Estrutura de página para site de cafeteria
* Layout organizado e responsivo
* Seções com informações sobre café e ambiente da cafeteria

## Como executar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
